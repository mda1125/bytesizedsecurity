{%- comment -%} 3-state Auto / Light / Dark theme toggle. Renders a single button that cycles Auto -> Light -> Dark -> Auto. Persists the explicit choice (Light or Dark) in localStorage under the key 'theme'; Auto is represented by the absence of a stored value. The no-FOUC bootstrap in head.html applies the resulting .dark class before first paint, so this include's JS is only responsible for the click handler, the live OS-pref listener, and keeping the visible icon / aria-label in sync. Three icon spans live inside the button; only the one matching the current state is visible (others get the `hidden` attribute). Uses `data-theme-toggle` attribute selectors so additional instances of the button (e.g., a duplicate in the mobile menu) Just Work. {%- endcomment -%}
